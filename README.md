# 🚲 Ride-Easy - Bike Rental Web Application

Ride-Easy is a responsive bike rental web application that allows users to browse available bikes, view vehicle details, and book bikes through a simple and user-friendly interface.

---

## 🚀 Features

- Responsive Design
- Browse Available Bikes
- Vehicle Details Page
- Online Booking Interface
- Contact Page
- Modern User Interface

---

## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Python
- Django

### Version Control
- Git
- GitHub
